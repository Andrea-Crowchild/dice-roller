This is just a simple dice roller I'm writing for myself. 
Syntax is "dice <no> <sides>" where <no> is the number of dice and
<sides> is the number of sides. Use "dice -h" to print the help dialogue.
Eventually this will have a testing suite to calculate the fairness of an
arbitrary number of rolls. 
Example:
dice 1 20
dice 5 6
Am going to be adding features like modifiers later.
